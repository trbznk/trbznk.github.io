Entwickle eine Klasse Bruch. Enhalten soll die Klasse:

Attribute

  • zaehler: int
  • nenner: int

Methoden

  • erweitern
  • kürzen
    
        >>> b = Bruch(2, 4)
        >>> b.erweitern(3)
        >>> b.zaehler, b.nenner
        (6, 12)
        >>> b = Bruch(3, 12)
        >>> b.kuerzen(3)
        >>> b.zaehler, b.nenner
        (1, 3)

Erweitere die Klasse um die Methode vereinfachen, die den Bruch so weit runterkürzt wie möglich.

    
        >>> b = Bruch(50, 40)
        >>> b.vereinfachen()
        >>> b.zaehler, b.nenner
        (5, 4)
